# Duplicate_names


## Installation

``` shell
# apt install required packages
sudo apt update
sudo apt install -y zip htop pipenv
  
# clone repo:
git cline https://github.com/SashaMogilevskii/duplicate_names.git

# create virtualenv:  
pipenv install

# activate virtualenv:  
pipenv shell

# run main.py
pipenv run python main.py

```
